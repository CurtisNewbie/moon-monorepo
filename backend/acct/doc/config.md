## Acct Configuration

| property                   | description                            | default value |
| -------------------------- | -------------------------------------- | ------------- |
| acct.category.builtin      | (`slice of objects`) built-in category |               |
| acct.category.builtin.code | built-in category code                 |               |
| acct.category.builtin.name | built-in category name                 |               |
